# React Interview Test

This task is to create a todo list app in react, the requirements are: 

- As a user I want to be able to see a list of items that i need to complete
- As a user I want to be able to add a new item to the to do list
- As a user I want to be able to remove an item from to do list
- As a user I want to be able to edit an item in the to do list
- As a user I want to be able to mark an item as done in the to do list
- As a user I should see a good user experience on mobile and tablet
- As a product owner I want the to do list application to be visually pleasing

## Pre setup

1. Created app with create-react-app
2. Installed redux
3. Installed redux dev tools
4. Installed node sass for sass support (however you can do styles however you prefer)
4. A default site reducer (not used in the app)
